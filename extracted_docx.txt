🔴  COMMON ERRORS & FIXES

FIX
Recharts PieChart center label not rendering
Recharts customLabel inside PieChart needs a renderCustomizedLabel function passed as label prop to <Pie>. Or use a positioned absolute <div> with CSS centering inside a relative wrapper.
Count-up animation flickers on re-poll
The animation should only run on mount (empty dependency array). On subsequent polls, update the value directly without retriggering the animation. Use a ref to track firstMount.
Grid overlaps on md breakpoint
Use grid-cols-2 md:grid-cols-3 xl:grid-cols-5 with gap-4. Ensure the donut chart wraps below on smaller screens using flex-wrap or a separate row.
fetch() call fails with CORS error
FastAPI must have CORSMiddleware with allow_origins=['*']. This is Rishi's backend — confirm it's running. For local dev add a Vite proxy: server.proxy in vite.config.js.

FIX
LineChart shows 'No data' or blank

Row click navigates but URL doesn't change

FIX
New alerts appear at bottom instead of top
Use setAlerts(prev => [...newAlerts, ...prev]) not [...prev, ...newAlerts]. Prepend new items.

BREAK-GLASS PROTOCOLS
Read before the hackathon. Not during.

If Things Go Wrong During the Demo

⚡  Backend API is down during presentation
IMMEDIATELY flip the DEMO toggle in the Sidebar. All pages swap to local mock data. The demo plays perfectly. Tell judges: 'We also built a full offline demo mode — let me show you how resilient the system is.'
⚡  Cytoscape canvas is blank
Call cy.resize() in a useEffect triggered by a button click. Or add a 'Reload Graph' button that destroys and recreates the Cytoscape instance. The data is there — only the render is broken.
⚡  React Flow diagram crashes (white screen)
Wrap the ReactFlow component in its own ErrorBoundary. The FSM diagram is a nice-to-have. If it crashes, hide it and show only the playbook cards — they still demonstrate the FSM concept.
⚡  Recharts throws on malformed data
Wrap all chart data access with optional chaining: stats?.trend_data ?? []. Never pass undefined to Recharts data prop. Always default to empty arrays.
⚡  App is blank — Vite build error
Run npm run dev and read the first error in the terminal. 99% of Vite errors are import path issues (wrong case, missing extension). Fix the import, the server hot-reloads instantly.
⚡  Less than 1 hour remaining, multiple pages incomplete
Priority 1 (non-negotiable): Overview + Live Alerts must work. Priority 2: Flip to DEMO mode and present all pages from mock data. Priority 3: Graph View and Vault can be shown as static screenshots if needed. The judges want to see the SOC concept — not just the React code.
